This repository contains all the C modules I validated during the 42 Piscine — a 4-week intensive coding bootcamp that serves as the entry selection process for 42 Lausanne.
Each module focuses on a specific set of concepts in C programming, validated through peer reviews and the **Moulinette** checker program.

> Final exam grade: **70% (Pass)**

---

## Overview of the modules

The 42 Piscine is structured around progressive modules, each introducing core concepts of the C language.

| Module | Main topics covered                                               |
|--------|-------------------------------------------------------------------|
| **C00** | Basic output, loops, conditionals, ASCII manipulation            |
| **C01** | Pointers, memory addresses, basic dereferencing                  |
| **C02** | String manipulation without standard library functions           |
| **C03** | Advanced string functions, searching, concatenation variants     |
| **C04** | Number conversion, recursion, custom bases                       |
| **C05** | Mathematics, iteration vs recursion, prime numbers               |
| **C06** | Command-line arguments (`argc`, `argv`), basic program structure |
| **C07** | Dynamic memory allocation (`malloc`, `free`), string splitting   |
| **C08** | Structures, header files, function pointers                      |

> Note: some rush/shell exercises are not included in this repository.

## 42 Norm and compilation

All code is written in the **42 Norm** and compiles with:

```bash
gcc -Wall -Wextra -Werror -o output file.c
```
